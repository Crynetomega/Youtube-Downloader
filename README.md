# Youtube-Downloader

This is a minimal web application to download YouTube clips as mp4 and mp3.

# Disclaimer

Downloading media of which you are not the copyright owner is against the YouTube Policy.

# Quick Start

### Install Node modules

```bash
npm init
```

### Run the application

```bash
npm run start
```
